# learn-to-play / Andean Duct Flutes (G)

Starter songbook for G-tuned Andean duct flutes from the build packet. Generated by the sheet-music skill.

This family covers duct-flute voices used for simple Andean-style pentatonic melodies and classroom ensemble parts.

Range: **G4 - G6 (two octaves). Pentatonic-first, with diatonic extensions where the build supports them.**

## Folder map
| Folder | Contents |
|---|---|

| `03-original/` | *High Pass Morning* - Heifer Zephyr original |

## Notes on notation
ABC files are concert-pitch practice sketches. Where a local flute has fewer holes, keep the pentatonic skeleton and omit passing tones.

## L2 readiness and known gaps
This starter songbook is ready for L2 review as a playable packet draft: ABC melodies, per-tune notes, static fingering summaries, warmups, and a top-level chart are present. Known gaps: generated audio, MusicXML, LilyPond, MIDI, and printable songsheets are skipped until the shared render dependencies are available; fingering charts need builder review against the final hole layout.
